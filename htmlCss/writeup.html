Master using '0' = Its very important to use "HOME" (use Esc-0) to rebound and to use "UP"(use Esc-o) goto the first letter on the row. Some of these quirks is what doesnt make people to want to be using the vim-key bindings. Using the '0' command is very important when one wants to visual select- This is so especially when you when you want to YANK, DEL, CUT AND PASTE - Hence, its very important to make '0' a muscle memory by practising often. Anything that may have to do with visually selecting should be done using '0' - followed by ALWAYS navigating to the very first line before pressing the 'v' - for selecting to the end before 'dd' - if you utilize any other ways aside of these it will mis-align your text. Moreso, making sure that one practise this often. Infact, anything that may have to do with the start of the sentence '0' should be used in line with 'w'
